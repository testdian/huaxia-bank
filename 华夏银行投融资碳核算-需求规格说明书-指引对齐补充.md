## 1. 监管工作程序（四阶段）

| 阶段 | 指引条款 | 系统模块 | 关键输出 |
|------|----------|----------|----------|
| 一、核算业务范畴 | 确定项目/非项目投融资业务类型 | 核算任务管理 | 任务、行业范围、余额口径 |
| 二、对象/边界/周期 | 识别核算对象、范围一/二、自然年 | 清单+正式清单+边界配置 | 正式清单、边界说明 |
| 三、数据采集与质控 | 按方法采集并验证 | 补数协同 | 信息采集表、附件 |
| 四、排放量计算 | 主体/项目排放+归因 | 计算引擎 | 单笔/汇总排放、DQR |

**组织要求：** 次年 **9月30日** 前完成本年度核算（可报备延期）。

## 2. 核算范畴与排除规则

### 2.1 纳入业务类型

- 项目贷款、固定资产贷款、流动资金贷款、贸易融资项下**贴现**等对公信贷

### 2.2 纳入行业（八大高碳）

电力、建材、钢铁、有色、石化、化工、造纸、民航（见 GB/T 4754 细分代码表，系统行业字典维护）。

### 2.3 排除规则（系统筛选必配）

| 规则编码 | 条件 |
|----------|------|
| LOW_BALANCE | 报告期内**月均融资额 < 500万元** |
| SME | 小型、微型企业 |
| INDIVIDUAL | 个人、个体工商户 |
| OVERSEAS | 融资主体在境外 |
| NON_HIGH_CARBON | 非八大高碳行业 |

## 3. 核算对象与边界

| 业务类型 | 核算对象 | 边界 | 周期 |
|----------|----------|------|------|
| 非项目投融资 | 融资主体 | 范围一（化石燃料、工艺排放）+ 范围二（外购电热） | 自然年度 |
| 项目投融资 | 项目（鼓励独立核算） | 项目建设/运营边界 | 自然年度 |

**字段（正式清单）：** 客户名称、业务类型(project/non_project)、核算对象类型、边界说明、scope1/scope2标识、核算周期。

## 4. 核算方法与计算公式

### 4.1 方法优先级（系统自动匹配，可人工确认）

1. **报告法**（质量等级 1）  
2. **物理活动法-能源法**（等级 2）  
3. **物理活动法-产品法**（等级 3）  
4. **经济活动法**（等级 4，营业收入或资产规模）  
5. **经济活动法-余额兜底**（等级 5，无法获取主体数据时）

### 4.2 主体排放量

| 方法 | 公式 |
|------|------|
| 报告法 | 直接采用披露/核查报告中的排放量 |
| 能源法 | E = Σ(E_i×EF_i) + E_process + E_elec×EF_elec + E_heat×EF_heat |
| 产品法 | E = Σ(P_k × EF_k) |
| 经济法 | E = 营业收入(或资产) × 行业排放因子 |

### 4.3 金融机构归因排放量（归集因子法）

| 类型 | 公式 |
|------|------|
| 非项目 | E业务 = E主体 × (投融资日均或月均余额 / 融资主体总资产) |
| 项目 | E业务 = E项目 × (项目融资日均或月均余额 / 项目总投资额) |
| 兜底 | E业务 = 投融资余额 × 行业排放因子 |

**余额口径可配置：** 日均 / 月均 / 年末（任务级配置）。

## 5. 数据采集字段清单（信息采集表）

### 5.1 企业基本信息

| 字段 | 说明 | 必填 |
|------|------|------|
| customerName | 企业名称 | 是 |
| creditCode | 统一社会信用代码 | 是 |
| gbIndustryCode | 国民经济行业代码 GB/T4754 | 是 |
| gbIndustryName | 行业名称 | 是 |
| industryMajor | 八大行业大类 | 是 |
| accountingYear | 核算年度 | 是 |
| totalAssets | 核算周期末总资产（万元） | 归因必填 |
| revenue | 主营业务收入（万元） | 经济法 |
| avgLoanBalance | 投融资日均/月均余额（万元） | 归因必填 |

### 5.2 报告法

| 字段 | 说明 |
|------|------|
| reportedEmission | 核算周期碳排放量(tCO2) |
| disclosureChannel | 披露渠道（年报/ESG/核查报告等） |
| thirdPartyVerified | 是否第三方核验 |

### 5.3 能源法 / 产品法 / 经济法

| 方法 | 核心字段 |
|------|----------|
| 能源法 | energyRows[] 或 energyTotalEmission；能源类型、消耗量、单位、因子 |
| 产品法 | productRows[] 或 productTotalEmission；产品、产量、因子 |
| 经济法 | economyBasis(revenue/assets)、economyValue、economyFactor |

### 5.4 附件

核查报告、ESG报告、能耗台账等（附件上传必留）。

## 6. 数据质量评价

### 6.1 单笔质量等级

| 等级 | 方法 |
|------|------|
| 1 | 报告法（第三方核验优先） |
| 2 | 能源法 |
| 3 | 产品法 |
| 4 | 经济法-收入 |
| 5 | 经济法-规模/余额兜底 |

### 6.2 机构整体 DQR

**公式：** DQR = Σ(单笔归因排放量 × 质量等级) / Σ(单笔归因排放量)

| DQR | 总体等级 |
|-----|----------|
| ≤1.5 | 优秀 |
| 1.5~2.0 | 良好 |
| 2.0~3.0 | 好 |
| 3.0~4.0 | 较好 |
| >4 | 一般 |
